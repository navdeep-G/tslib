# Phase 11: release-candidate hardening

Phase 11 focuses on trust, reproducibility, and contributor ergonomics rather than adding new model classes.

## Included work

- stronger regression tests for edge cases and end-to-end workflows
- benchmark markdown export helper and benchmark-results template
- README model chooser and explicit testing section
- package-level Javadocs for primary public packages
- CI expanded to Java 17 and Java 21, plus nightly verification

## Goal

Make the library easier to release, easier to evaluate, and safer to change.
